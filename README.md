# 上班修仙傳

> 練氣不如練肝，飛升不如下班。

一款詼諧的上班族修仙放置網頁遊戲。純前端（HTML + CSS + Vanilla JS），無後端、無資料庫、無框架——丟到任何靜態主機（cPanel、GitHub Pages、Netlify）就能玩。

🎮 **線上試玩**：<https://tohohi.com/game/>

## 遊戲特色

- **修仙 × 職場**：練氣期＝無薪實習生，飛升＝財富自由退休。上班累積修為與靈石，心魔太重會突破失敗（走火入魔＝過勞）
- **水見式入職測驗**：握住公司咖啡判定六大流派——爆肝系、外包系、話術系、管理系、簡報系、天選系（致敬獵人的念能力系統）
- **體力制**：行動消耗體力，體力隨「現實時間」恢復，關掉頁面去休息也算修行
- **劇情副本**：升級獲得副本道具，代替陷入危機的掌門出戰——骰子檢定、劇情分歧、寶可夢式回合制戰鬥（含流派專屬技與「擲骰命運」）
- **假課金福利社**：每日「儲值 NT$0」、裝備轉蛋（N/R/SR/SSR）、+9 領帶強化系統。強化失敗不會損壞裝備——上班已經很苦了
- **三萬多字劇情**：六位掌門各六章個人故事、十一章主線、每日老闆訓話、人物名冊檔案解鎖
- **玩家自訂**：上傳自己的大頭照當主角、道號命名、存檔碼跨裝置備份

## 部署

把整個資料夾放上任何靜態網頁空間即可。存檔使用瀏覽器 localStorage。

```
game/
├── index.html    # 遊戲本體（引擎 + UI）
├── story.js      # 劇情資料（對話、章節、名冊檔案）
├── dungeons.js   # 劇情副本資料（節點圖：say / choice / dice / battle / end）
└── img/
    ├── avatar/   # 角色大頭
    └── bg/       # 場景底圖（demo 圖，可自行替換同檔名覆蓋）
```

可調參數（`index.html` 開頭）：

| 參數 | 預設 | 說明 |
|---|---|---|
| `REGEN_SEC` | 5 | 每 N 秒回復 1 點體力（示範速度，正式建議 60+） |
| `TASK_INTERVAL_MS` | 3 小時 | 阿慶發放派工任務的間隔 |

## 改劇情

所有文本都在 `story.js`（對話、主線、訓話）與 `dungeons.js`（副本腳本），純資料格式，不用碰遊戲邏輯。對話內的 `{name}` 會替換成玩家道號。
